## 项目介绍

WP Clean Admin 是一个功能强大的 WordPress 插件，旨在帮助管理员管理和优化 WordPress 后台，提高网站性能和管理效率。
## 功能特点

### 1. 后台清理
- 删除不必要的 WordPress 后台菜单
- 隐藏不必要的仪表盘小工具
- 清理后台头部和底部的多余信息
- 优化登录页面样式

### 2. 性能优化
- 禁用不必要的 WordPress 功能
- 优化数据库查询
- 减少 HTTP 请求
- 优化资源加载

### 3. 安全增强
- 限制登录尝试次数
- 隐藏 WordPress 版本信息
- 禁用 XML-RPC（可配置）
- 增强后台安全设置

### 4. 用户权限管理
- 自定义用户角色和权限
- 限制用户访问特定功能
- 优化用户管理界面

### 5. 菜单定制
- 自定义后台菜单排序
- 创建自定义菜单组
- 隐藏特定用户的菜单

### 6. 数据库管理
- 数据库清理和优化
- 备份和恢复数据库
- 优化数据库表结构

## 安装方法

### 方法 1：直接上传
1. 下载插件的 ZIP 压缩包
2. 登录 WordPress 后台，进入"插件"→"添加插件"
3. 点击"上传插件"，选择下载的 ZIP 包
4. 点击"现在安装"，然后点击"激活插件"

### 方法 2：FTP 上传
1. 解压插件的 ZIP 压缩包
2. 使用 FTP 客户端连接到您的网站
3. 将解压后的文件夹上传到 `/wp-content/plugins/` 目录
4. 登录 WordPress 后台，进入"插件"→"已安装插件"
5. 找到"WP Clean Admin"，点击"激活"

### 方法 3：GitHub 克隆
```bash
git clone https://github.com/sutchan/WP-Clean-Admin.git wp-content/plugins/wp-clean-admin
```

## 使用说明

1. 安装并激活插件后，在 WordPress 后台左侧菜单会出现"WP Clean Admin"选项
2. 点击进入插件设置页面，根据需要配置各项功能
3. 配置完成后，点击"保存设置"
4. 插件会自动应用您的设置，优化 WordPress 后台

## 常见问题

### Q: 插件会影响我的网站性能吗？
A: 不会，插件的设计目的就是优化网站性能，它只会删除不必要的功能，不会增加额外的负担。

### Q: 插件支持多语言吗？
A: 是的，插件支持多语言，目前已包含英文和中文版本。

### Q: 如何恢复默认设置？
A: 在插件设置页面底部，点击"恢复默认设置"按钮即可。

### Q: 插件与其他插件冲突吗？
A: 插件经过测试，与大多数主流 WordPress 插件兼容。如果您遇到冲突问题，请通过 GitHub Issues 联系我们。
