# WP Clean Admin\r\n\r\n一个全面的 WordPress 后台清理和优化插件。\r\n\r\n## 项目介绍\r\n\r\nWP Clean Admin 是一个功能强大的 WordPress 插件，旨在帮助管理员清理和优化 WordPress 后台，提高网站性能和管理效率。\r\n\r\n## 功能特点\r\n\r\n### 1. 后台清理\r\n- 删除不必要的 WordPress 后台菜单\r\n- 隐藏不需要的仪表盘小工具\r\n- 清理后台顶部和底部的多余信息\r\n- 优化登录页面样式\r\n\r\n### 2. 性能优化\r\n- 禁用不必要的 WordPress 功能\r\n- 优化数据库查询\r\n- 减少 HTTP 请求\r\n- 优化资源加载\r\n\r\n### 3. 安全增强\r\n- 限制登录尝试次数\r\n- 隐藏 WordPress 版本信息\r\n- 禁用 XML-RPC（可配置）\r\n- 增强后台安全设置\r\n\r\n### 4. 用户权限管理\r\n- 自定义用户角色和权限\r\n- 限制用户访问特定功能\r\n- 优化用户管理界面\r\n\r\n### 5. 菜单定制\r\n- 自定义后台菜单排序\r\n- 创建自定义菜单组\r\n- 隐藏特定用户的菜单\r\n\r\n### 6. 数据库管理\r\n- 数据库清理和优化\r\n- 备份和恢复数据库\r\n- 优化数据库表结构\r\n\r\n## 安装方法\r\n\r\n### 方法 1：直接上传\r\n1. 下载插件的 ZIP 压缩包\r\n2. 登录 WordPress 后台，进入"插件"→"添加插件"\r\n3. 点击"上传插件"，选择下载的 ZIP 包\r\n4. 点击"现在安装"，然后点击"激活插件"\r\n\r\n### 方法 2：FTP 上传\r\n1. 解压插件的 ZIP 压缩包\r\n2. 使用 FTP 客户端连接到您的网站\r\n3. 将解压后的文件夹上传到 `/wp-content/plugins/` 目录\r\n4. 登录 WordPress 后台，进入"插件"→"已安装插件"\r\n5. 找到"WP Clean Admin"，点击"激活"\r\n\r\n### 方法 3：GitHub 克隆\r\n```bash\r\ngit clone https://github.com/sutchan/WPCleanAdmin.git wp-content/plugins/wp-clean-admin\r\n```\r\n\r\n## 使用说明\r\n\r\n1. 安装并激活插件后，在 WordPress 后台左侧菜单会出现"WP Clean Admin"选项\r\n2. 点击进入插件设置页面，根据需要配置各项功能\r\n3. 配置完成后，点击"保存设置"\r\n4. 插件会自动应用您的设置，优化 WordPress 后台\r\n\r\n## 系统要求\r\n\r\n- WordPress 5.0 或更高版本\r\n- PHP 7.0 或更高版本\r\n- MySQL 5.6 或更高版本\r\n\r\n## 许可证\r\n\r\nWP Clean Admin 使用 GPL-2.0+ 许可证，详情请查看 [LICENSE](https://github.com/sutchan/WPCleanAdmin/blob/main/LICENSE) 文件。\r\n\r\n## 联系方式\r\n\r\n- 项目地址：https://github.com/sutchan/WPCleanAdmin\r\n- 作者：Sut\r\n- 邮箱：请通过 GitHub Issues 联系\r\n\r\n## 版本历史\r\n\r\n请查看 [CHANGELOG](CHANGELOG.md) 文件了解版本更新历史。
