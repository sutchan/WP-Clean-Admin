# WP Clean Admin

一个全面的 WordPress 后台清理和优化插件。

## 项目介绍

WP Clean Admin 是一个功能强大的 WordPress 插件，专注于帮助管理员清理和优化 WordPress 后台，提升网站性能和管理效率。

## 功能特点

### 1. 后台清理
- 删除不必要的 WordPress 后台菜单
- 隐藏不需要的仪表盘小工具
- 清理后台头部和底部的冗余信息
- 优化登录页面样式

### 2. 性能优化
- 禁用不必要的 WordPress 功能
- 优化数据库查询
- 减少 HTTP 请求
- 优化资源加载

### 3. 安全增强
- 限制登录尝试次数
- 隐藏 WordPress 版本信息
- 禁用 XML-RPC（可配置）
- 增强后台安全设置

### 4. 用户权限管理
- 自定义用户角色和权限
- 限制用户访问特定功能
- 优化用户管理界面

### 5. 菜单定制
- 自定义后台菜单排序
- 创建自定义菜单组
- 隐藏特定用户的菜单

### 6. 数据库管理
- 数据库清理和优化
- 备份和恢复数据库
- 优化数据库表结构

## 安装方法

### 方法 1：直接上传
1. 下载插件的 ZIP 压缩包
2. 登录 WordPress 后台，进入"插件"→"添加插件"
3. 点击"上传插件"，选择下载的 ZIP 包
4. 点击"现在安装"，然后点击"激活插件"

### 方法 2：FTP 上传
1. 解压插件的 ZIP 压缩包
2. 使用 FTP 客户端连接到您的网站
3. 将解压后的文件夹上传到 `wp-content/plugins/` 目录
4. 登录 WordPress 后台，进入"插件"→"已安装插件"
5. 找到"WP Clean Admin"，点击"激活"

### 方法 3：GitHub 克隆
```bash
git clone https://github.com/sutchan/WP-Clean-Admin.git wp-content/plugins/wp-clean-admin
```

## 使用说明

1. 安装并激活插件后，在 WordPress 后台左侧菜单会出现"WP Clean Admin"选项
2. 点击进入插件设置页面，根据需要配置各页面功能
3. 配置完成后，点击"保存设置"
4. 插件会自动应用保存的设置，优化 WordPress 后台

## 系统要求

- WordPress 5.0 或更高版本
- PHP 7.0 或更高版本
- MySQL 5.6 或更高版本
