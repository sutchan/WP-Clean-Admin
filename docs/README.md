是一个功能强大的 WordPress 插件，旨在帮助管理员管理和优化 WordPress 后台，提高网站性能和管理效率。\r\n## 功能特点\r\n\r\n### 1. 后台清理\r\n- 删除不必要的 WordPress 后台菜单\r\n- 隐藏不必要的仪表盘小工具\r\n- 清理后台头部和底部的多余信息\r\n- 优化登录页面样式\r\n\r\n### 2. 性能优化\r\n- 禁用不必要的 WordPress 功能\r\n- 优化数据库查询\r\n- 减少 HTTP 请求\r\n- 优化资源加载\r\n\r\n### 3. 安全增强\r\n- 限制登录尝试次数\r\n- 隐藏 WordPress 版本信息\r\n- 禁用 XML-RPC（可配置）\r\n- 增强后台安全设置\r\n\r\n### 4. 用户权限管理\r\n- 自定义用户角色和权限\r\n- 限制用户访问特定功能\r\n- 优化用户管理界面\r\n\r\n### 5. 菜单定制\r\n- 自定义后台菜单排序\r\n- 创建自定义菜单组\r\n- 隐藏特定用户的菜单\r\n\r\n### 6. 数据库管理\r\n- 数据库清理和优化\r\n- 备份和恢复数据库\r\n- 优化数据库表结构\r\n\r\n## 安装方法\r\n\r\n### 方法 1：直接上传\r\n1. 下载插件的 ZIP 压缩包\r\n2. 登录 WordPress 后台，进入"插件"→"添加插件"\r\n3. 点击"上传插件"，选择下载的 ZIP 包\r\n4. 点击"现在安装"，然后点击"激活插件"\r\n\r\n### 方法 2：FTP 上传\r\n1. 解压插件的 ZIP 压缩包\r\n2. 使用 FTP 客户端连接到您的网站\r\n3. 将解压后的文件夹上传到 `/wp-content/plugins/` 目录\r\n4. 登录 WordPress 后台，进入"插件"→"已安装插件"\r\n5. 找到"WP Clean Admin"，点击"激活"\r\n\r\n### 方法 3：GitHub 克隆\r\n```bash\r\ngit clone https://github.com/sutchan/WPCleanAdmin.git wp-content/plugins/wp-clean-admin\r\n```\r\n\r\n## 使用说明\r\n\r\n1. 安装并激活插件后，在 WordPress 后台左侧菜单会出现"WP Clean Admin"选项\r\n2. 点击进入插件设置页面，根据需要配置各项功能\r\n3. 配置完成后，点击"保存设置"\r\n4. 插件会自动应用您的设置，优化 WordPress 后台\r\n\r\n## 配置选项\r\n\r\n### 常规设置\r\n- 启用/禁用插件功能\r\n- 设置插件语言\r\n- 配置日志记录\r\n\r\n### 后台清理\r\n- 选择要隐藏的菜单\r\n- 选择要隐藏的仪表盘小工具\r\n- 配置后台头部和底部信息\r\n\r\n### 性能优化\r\n- 选择需要的 WordPress 功能\r\n- 配置数据库优化选项\r\n- 配置资源加载优化\r\n\r\n### 安全设置\r\n- 配置登录尝试限制\r\n- 启用/禁用 XML-RPC\r\n- 配置安全头部信息\r\n\r\n### 用户权限\r\n- 自定义用户角色\r\n- 分配用户权限\r\n- 限制用户访问\r\n\r\n## 常见问题\r\n\r\n### Q: 插件会影响我的网站性能吗？\r\nA: 不会，插件的设计目的就是优化网站性能，它只会删除不必要的功能，不会增加额外的负担。\r\n\r\n### Q: 插件支持多语言吗？\r\nA: 是的，插件支持多语言，目前已包含英文和中文版本。\r\n\r\n### Q: 如何恢复默认设置？\r\nA: 在插件设置页面底部，点击"恢复默认设置"按钮即可。\r\n\r\n### Q: 插件与其他插件冲突吗？\r\nA: 插件经过测试，与大多数主流 WordPress 插件兼容。如果您遇到冲突问题，请通过 GitHub Issues 联系我们。\r\n\r\n## 系统要求\r\n\r\n- WordPress 5.0 或更高版本\r\n- PHP 7.0 或更高版本\r\n- MySQL 5.6 或更高版本\r\n\r\n## 贡献指南\r\n\r\n欢迎您为 WP Clean Admin 做出贡献，请查看
