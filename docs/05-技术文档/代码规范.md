PHP 代码规范\r\n\r\n### 2.1 基础语法规范\r\n- **PHP 标签**：使用 `<?php` 标签，禁止使用短标签 `<?`\r\n- **闭合标签**：PHP 文件末尾禁止使用闭合标签 `?>`\r\n- **命名空间**：使用 `WPCleanAdmin` 命名空间，例如 `namespace WPCleanAdmin;`\r\n- **自动加载**：遵循 PSR-4 修饰符\r\n- **方法参数**：参数类型声明，返回值类型声明\r\n- **文档注释**：每个类和方法必须包含 PHPDoc 注释\r\n\r\n**示例代码**：\r\n```php\r\n<?php\r\n/**\r\n * Core class for WP Clean Admin plugin\r\n *\r\n * @package WPCleanAdmin\r\n `$_GET`、`$_POST`、`$_SERVER`\r\n\r\n### 2.4 函数设计规范\r\n- **函数命名**：使用 `function` 关键字，函数名使用 snake_case\r\n- **引用传递**：明确使用 `&` 符号表示引用传递\r\n- **文档注释**：每个函数必须包含 PHPDoc 注释\r\n- **错误处理**：使用 `WP_Error` 类处理错误\r\n\r\n**示例代码**：\r\n```php\r\n/**\r\n * Get plugin settings\r\n *\r\n * @param string $key Specific setting key to get\r\n * @param mixed $default Default value if setting not found\r\n * `!=`\r\n\r\n**示例代码**：\r\n```php\r\n// 条件语句\r\nif ( $a === $b ) {\r\n    // 执行代码\r\n} elseif ( $a > $b ) {\r\n    // 执行代码\r\n} else {\r\n    // 执行代码\r\n}\r\n\r\n// 循环语句\r\nforeach ( $array as $key => 函数处理 SQL 查询\r\n- **XSS 防护**：使用 `esc_html()`、`esc_attr()`、`esc_url()` 等函数转义输出\r\n- **CSRF 防护**：使用 `wp_nonce_field()` 和 `wp_verify_nonce()` 函数防止 CSRF 攻击\r\n- **权限检查**：使用 `wpca_current_user_can()`
